== Description ==
Mymagazine is a creative and responsive WordPress theme. It has a clean and colorful magazine style, perfect for news websites dedicated to sports or travels.

For more information about MyMagazine please go to http://www.svgthemes.com/mymagazine

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in MyMagazine in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Go to http://www.svgthemes.com for a guide on how to customize this theme.
5. Navigate to Appearance > Customize in your admin panel and customize to taste.
